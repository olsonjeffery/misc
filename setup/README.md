# Omarchy setup

## Strip branding

### Change screensaver

- Install->Package->figlet
- Install->AUR->figlet-fonts
- Install->AUR->figlet-fonts-extra
- pick a font: bigmono12, smmono12 or smmono9
- write out a logo to a file named `logo.txt`
- `mv logo.txt ~/.config/omarchy/branding/screensaver.txt`
